class OrderInitialized(TraderId trader_id, StrategyId strategy_id, InstrumentId instrument_id, ClientOrderId client_order_id, OrderSide order_side, OrderType order_type, Quantity quantity, TimeInForce time_in_force, bool reduce_only, dict options, OrderListId order_list_id, ClientOrderId parent_order_id, list child_order_ids, ContingencyType contingency, list contingency_ids, unicode tags, UUID4 event_id, int64_t ts_init)¶

Bases: nautilus_trader.model.events.order.OrderEvent

Represents an event where an order has been initialized.

This is a seed event which can instantiate any order through a creation method. This event should contain enough information to be able to send it ‘over the wire’ and have a valid order created with exactly the same properties as if it had been instantiated locally.
