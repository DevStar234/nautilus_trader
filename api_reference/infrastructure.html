Cache¶

class RedisCacheDatabase(TraderId trader_id, Logger logger, InstrumentSerializer instrument_serializer, CommandSerializer command_serializer, EventSerializer event_serializer, dict config)¶

Bases: nautilus_trader.cache.database.CacheDatabase

Provides a cache database backed by Redis.

Warning

Redis can only accurately store int64 types to 17 digits of precision. Therefore nanosecond timestamp int64’s with 19 digits will lose 2 digits of precision when persisted. If precision to this level is important, then you could additionally persist events in another medium/database which can properly handle int64 types.
